# ML_demo
This is basic repo to understand GIT command and CI/CD.

creating conda environment
...
conda create -p venv python==3.7 -y
...
conda activate venv/
...

To add files to git 
...
git add .
OR 
git add filename

To check all versions
...
git log

To solve error of task running 
...
rm -f .git/index.lock
